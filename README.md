## О приложении

**Happ** — это приложение для управления прокси-подключениями, построенное на базе ядра **Xray**. Оно позволяет быстро создавать новые конфигурации, импортировать уже существующие, редактировать параметры подключения и удобно переключаться между профилями.

Программа поддерживает современные сетевые технологии и гибкую систему маршрутизации, а интуитивно понятный интерфейс делает ее комфортной как для новичков, так и для опытных пользователей.

## Основные возможности

* Создание, импорт и редактирование прокси-конфигураций.
* Гибкая настройка правил маршрутизации трафика.
* Поддержка популярных протоколов:

  * **VLESS (Reality)**
  * **VMess**
  * **Trojan**
  * **Shadowsocks**
  * **SOCKS**
* Быстрое переключение между сохраненными профилями.
* Управление всеми конфигурациями из одного приложения.

## Конфиденциальность

Happ придерживается принципов максимальной приватности. Приложение не собирает статистику использования, не передает конфигурации на удаленные серверы и не отправляет персональные данные третьим лицам. Все профили, настройки и служебные данные сохраняются исключительно на устройстве пользователя.

## Важная информация

Happ представляет собой исключительно инструмент для работы с прокси-конфигурациями. Приложение **не предоставляет VPN-сервис, прокси-серверы, готовые подключения или платные подписки**. Для использования необходимо иметь собственный сервер либо воспользоваться услугами стороннего провайдера. Пользователь самостоятельно несет ответственность за соблюдение законодательства своей страны при эксплуатации приложения.
